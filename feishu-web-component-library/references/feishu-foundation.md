# 飞书 Web 设计基础（Foundation）

## 1. 设计原则（跨文档汇总）
- 优先保证信息效率：在批量浏览/批量操作场景可提高密度；在警示、关键决策、触控场景保持宽松密度。
- 统一层级表达：通过颜色、字号、圆角、描边、投影共同表达信息层级，避免同层级多种视觉语言混用。
- 语义优先于视觉：按钮/标签/图标先满足语义和可理解性，再做视觉强调。
- 多端一致+按端适配：保持语义一致，允许在断点下做组件映射和能力降级。

## 2. 色彩体系

### 2.1 品牌与功能色
- 品牌主色：`#1456F0`（行动号召、选中、高亮、链接等）。
- 功能色（状态语义）：
  - 信息 `#4275E0`
  - 成功 `#4BA254`
  - 警告 `#E57A3C`
  - 危险 `#E55E55`
- 辅助分类色板：用于图表与分类表达，分类色板建议按固定顺序使用；类别 > 14 时循环取色。

### 2.2 中性色（文字/图标/背景/边框）
- 文本：`#1F2329` / `#646A73` / `#8F959E` / `#BBBFC4`
- 交互描边：`#D0D3D6`
- 卡片描边：`#DEE0E3`
- 禁用填充：`#EFF0F1`
- 背景层：`#EFF0F1` / `#F2F3F5` / `#F5F6F7`
- 遮罩：`rgba(0,0,0,0.55)`

### 2.3 透明度梯度（图片标注复核）
- 深浅背景都使用统一梯度：`5% 10% 15% 20% 30% 40% 50% 60% 69% 80% 89%`
- 透明度仅建议叠加在 `#1F2329` 或 `#FFFFFF` 两条主轴上。

## 3. 字体与排版

### 3.1 字重
- Regular `400`：正文主用
- Medium `500`：标题与中高优先级文本
- Semibold `600`：数字大标题/强调标题

### 3.2 字阶（从图文规范提炼）
- `26/40 semibold`：特大标题
- `24/36 semibold`：一级标题
- `20/30 medium`：二级标题
- `17/26 medium`：三级标题
- `16/24 regular`：正文
- `14/22 regular|medium`：正文辅助
- `12/20 regular|medium`：说明文字
- `10/16 regular|medium`：极小辅助

### 3.3 文本规则
- 正文强调可用 medium，不建议频繁使用 semibold 造成噪音。
- 多语言场景下优先调整布局而非强行截断。

## 4. 栅格、间距与布局

### 4.1 基础网格
- Web/桌面：以 `8px` 为基准，局部可使用 `4px`。
- Mobile（@1x）：以 `4pt/dp` 为基准，局部 `2pt/dp`。

### 4.2 断点与列
- `XS: 0-599`（4 列，Gutter 8，Margin 16）
- `S: 600-1023`（8 列，Gutter 8，Margin 16）
- `M: 1024-1439`（12 列，Gutter 16，Margin 24）
- `L: 1440-1919`（12 列，Gutter 16，Margin 24）
- `XL: >=1920`（12 列体系延续）

### 4.3 约束
- 栅格优先等分，避免 5/7 等非整除分栏成为主布局规则。
- 安全边距通用最小值 `16px`；业务可在同体系内自定义（如 20/24）但需全局一致。
- 定宽页面建议内容极限宽度 `1136px`。

## 5. 圆角、描边、投影

### 5.1 圆角
- `Radius-XS = 4`：原子组件（复选框、Tag）
- `Radius-S = 6`：按钮、输入框、下拉菜单、提示
- `Radius-M = 8`：对话框、气泡卡片、上传卡片
- `Radius-L = 10`：大卡片/复杂容器
- `Radius-XL = 全圆角`：圆形徽标、全圆角按钮、悬浮按钮

### 5.2 描边
- 交互控件描边：1px `#D0D3D6`
- 卡片描边：1px `#DEE0E3`
- 列表/Feed 分割线：1px（部分端 0.5px）

### 5.3 投影
- 方向语义：`down/up/left/right` 由容器来源方向决定。
- 推荐轻量阴影，不与重描边叠加造成视觉过重。
- 主色强调阴影可使用蓝色阴影（如 `#245BDB` 系列透明度）。

## 6. 图标规范
- 统一栅格：24x24 作为基础绘制网格。
- 常用输出尺寸：16/20/24/32。
- 优先线性图标，主线宽推荐 `2px`。
- 图标按钮建议配 Tooltip，避免语义不明。

## 7. 动效规范
- 基础效果：位移、缩放、旋转、透明度、颜色。
- 规律：变化小用短时长，变化大用长时长，退出通常比进入更短。
- 标准曲线：`cubic-bezier(0.3, 0, 0.1, 1)`（图中参数示例）。
- 线性曲线：用于透明度/颜色等均匀过渡。
- 交互动效交付需标明：触发条件、对象、属性、曲线、时间轴。

## 8. 默认 Token 清单（实现建议）
- 直接采用：`assets/templates/theme.default.json`
- 高密度模式：`assets/templates/theme.compact.json`
